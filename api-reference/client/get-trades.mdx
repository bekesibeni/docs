# GET /client/trades

Returns the trade history for the authenticated client. Supports filtering by type and game, plus pagination.

**Authentication:** Client Token (`Authorization` header)

## Request

```http
GET https://api.assetpay.co/client/trades?type=DEPOSIT&game=730&page=1&limit=10
Authorization: CLIENT_TOKEN
```

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `type` | string | - | Filter by `"DEPOSIT"` or `"WITHDRAW"` |
| `game` | string | `730` | Game app ID |
| `page` | number | `1` | Page number |
| `limit` | number | `10` | Trades per page |
